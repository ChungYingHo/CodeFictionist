開始之前先來看看[植宇宙](https://plantae-tw.vercel.app/)的形象網站首頁 logo：  
![](./assets/mask-result.png)

當我第一次做出這個圖片遮罩效果，有一些同行或朋友問我：這是後製？還是用什麼工具做出來的？  
其實這是用 CSS 的 mask 屬性做出來的，這個屬性可以讓我們在網頁上做出各種圖片遮罩效果。

具體來說上面那個圖片 logo 的原型是這兩張圖：
1. 植宇宙 ：  
![](./assets/mask-logo.png)
2. 小麥背景圖：  
![](./assets/mask-bg.jpg)

透過下列 code 就可以做出上面的效果：
```html
<div class="mask" ></div>
```
```css
.mask {
  /* highlight-start */
  background: url('/plantae/index.jpg') no-repeat top center;
  background-size: cover;
  -webkit-mask: url('/plantae/logo.png') no-repeat center center;
  mask: url('/plantae/logo.png') no-repeat center center;
  -webkit-mask-size: contain;
  mask-size: contain;
  /* highlight-end */
  width: 16rem;
  height: 16rem;
  display: flex;
  justify-content: center;
  align-items: center;
  border-radius: 50%;
}
```

可以看到，一開始是把元素的背景設為小麥背景圖，之後透過 `-webkit-mask` 和 `mask` 這兩個屬性，把 logo 圖片設為遮罩，最後再設定 `mask-size` 來調整遮罩的大小。
