叫 **30 Days Series** 了，是因為這一專區原本目的是為了 **iThome 鐵人賽** 設置的。  
打從我剛成為一個前端工程師時，就下定決心總有一天也要參與看看這個台灣軟體界的年度盛事，可以的話，希望能跟大神們一樣，寫著寫著就出書了 :relieved:。

雖然是 30 天系列，但我尋思著，我大概也不會每年都參加鐵人塞，畢竟體驗一次後就會覺得這種連續一個月學技術、寫 code、寫文章的生活其實非常 hard core。我的一位團隊夥伴 Chloe 跟我說，她可以一天花 2 小時學習 UI/UX，然後再花 N 小時實作 + 寫文章。  
所以呢，雖然這裡標題是 **30 Days Series**，但我覺的更可以定調微我給自己一個主題的挑戰紀錄空間。

什麼主題呢？我有想過試試 [JavaScript 30](https://javascript30.com/) 或 [Frontend Mentor](https://www.frontendmentor.io/)，但老實說，我自己是個閒不下來的人，這不，又奔著 C# 跟 Unity 去了。  
所以這裡之後出現的主題可能會出現那種跟 Youtube 標題一樣的什麼的：**一週 2D 遊戲開發挑戰**、**兩週教你學會用 Vue 做一個購物網站**之類的鬼東西 :smiley:

Well，我是真的不知道這個專區未來會怎麼發展啦，說不定下次這個專區的新文章是什麼 203X iThome 鐵人賽也不一定呢 :smirk:。  
大家就權當開驚喜包一樣來看這個專區吧~
