本來想 30 天以一個全端專案為主題來寫的，但後來思考了好久，這種綜觀前端、後端、資料庫的專案內容實在過於龐大，以 Nuxt 來做全端開發也要考量 JavaScript (或 TypeScript)、Vue、SQL、CSS 與資料庫等技術，這要 30 天完成，抱歉我不是大神，我做不到 :sweat_smile:。

所以後來想想，我自己在做 side project 時，其實滿常玩一些奇怪的 CSS 的特效的，但那些特效往往都沒人整合，都要一篇篇去 Google，那我何不透過這次 30 天來整理並學習一下呢？  
所以呢，這次 2024 鐵人賽主題就定在 **30 天 CSS 特效挑戰**吧。
