## 在 CSS 裡相對跟絕對很重要
我會說「相對」與「絕對」是玩轉 CSS 中兩個很熟記的單詞，怎麼說呢，除了在尺寸單位這裡等等會出現之外，大家很耳熟能詳的 position 們也是基於相對跟絕對的概念，還記得吧，`position: absolute` 參照的是最近的有 `position: relative` 的父元素，而 `position: fixed` 則是參照的是整個瀏覽器視窗。  
好，如果不記得那就算了 XD  
position 不是這裡要探討的重點。我們要釐清的是，「相對」這個詞在 CSS 中是指會對到誰的設定，而這個誰通常是父元素；而「絕對」通常是指直接對到整個根元素。
