大年初一，開箱 4070s！

January 29, 2025 · 5 min read
Chung-Ying
菜鳥前端工程師

今天大年初一，1 月 29 日，明天就是 50 系列顯示卡的發售日，然後我卻在昨天除夕時收到新電腦 www
對，我沒組 50 系列，在經過一個多月的激烈洗腦討論後，我成功被朋友勸退了 XD
所以這是一篇紀錄我被勸退的文章及新電腦的開箱文~

電腦規格​

  • CPU: AMD Ryzen 7 7700
  • 主機板：GigaByte B650 AOROUS ELITE AX V2
  • RAM: 十銓 T-FORCE DELTA RGB DDR5-6400 16GB x 2
  • SSD: Solidigm P44 Pro 2TB + KIOXIA Exceria G2 2TB (顯卡送的)
  • 顯示卡：GigaByte RTX 4070 SUPER EAGLE OC 12GB
  • 電源供應器：MONTECH TITAN 750W
  • 塔扇：Jonsbo CR-3000 黑
  • 機殼：聯力 Lancool 216 ARGB 黑

新電腦設定​

組桌機跟買筆電的差異是筆電匯預先裝好一些軟體和設定，但桌機就算請店家先幫忙裝好 windows 系統，回來後還是要自己設定一些東西。以下紀錄一下這台電腦我拿到後進行了一些主要設定。

  1. wifi：我一開始拿到電腦想說怎麼收到的 wifi 訊號那麼薄弱，後來才發現這張主機板有提供天線，裝上去就一切順利了。
  2. windows：我有買隨機版病癒先請店家幫我安裝好系統，但實際上拿到電腦後還是得翻一下箱子找到 windows 金鑰。
  3. 防毒軟體：經過卡巴斯基跟 Norton，這一次我決定使用 Windows Defender。
  4. 通訊軟體：Discord 跟 Line。這裡被 Line 雷到，Microsoft Store 裡的 Line 是舊版的，要去官網下載才是 PC 新版。
  5. 遊戲：組 4070s 就是為了打遊戲啊，所以裝了 HoYoPlay 跟 Steam。
  6. 程式相關：nvm、Node.js、VScode、Visual Studio、Unity。一切安照我之前寫的環境建置文順利進行。
info

新專案執行 npm install 有權限問題的話可以執行下列指令：

Set-ExecutionPolicy -Scope CurrentUser

之後輸入參數 remotesigned 即可。

螢幕決定顯卡論​

好了，這感覺才是這篇的重點 XD
所謂螢幕決定顯卡論是我朋友把我從 50 系列顯卡勸退的論點，我覺得實在太有道理了，忍不住想寫篇文章紀錄一下 www

事情是這樣的，1 月 30 日 5080、5090 將要發售，我本來是打算組 5080 的，喔，那個預算齁，拉一拉，5080 組整機大概要 70000 元，本來想著大年初二直接年終給它刷下去，然後在大概一週前，我朋友問了我幾個問題：

  1. 螢幕大小
  2. 2K 還是 4K

我的螢幕是 27 吋華碩 TUF VG27AAQL3A 的電競 2K 螢幕，對我來說 27 吋真的夠大了，所以我也沒想過要換更大的螢幕，所以他的論點是：

  1. 2K 跟 4K 在 32 吋以下的螢幕上差異不大。
  2. 使用 2K 螢幕的話，根本用不到 5080。
  3. 50 系列顯卡可能需要升級主機板，而且 5070ti、5070 跟 4070s 在 2K 螢幕上的效能差異不大，都屬於可以跑高幀、極高畫質的等級。
  4. 4070s 大概還可以打五年內的 3A 遊戲。
  5. 可以省 2-30000 元 XD

後來想一想齁，我之前試看 2K 跟 4K 我是真的很難看到差異啦，螢幕也沒想換，基於這樣的理由，我就實際一點組了 4070s，剛好也可以撿一些高 CP 值的組合順便幫店家清庫存
以上，就是後來組 4070s 的理由 XD
