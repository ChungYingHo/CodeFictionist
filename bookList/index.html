這頁其實只是我自己想看的書的書單~

前言​

就像 steam 是買遊戲的遊戲，博客來是買書的遊戲...好，這譬喻有點不恰當，我想表達的其實是我買了一堆書都沒有看😣
老實說，我至今到底買了哪些書沒看我也不太清楚了，有時候就是書店逛著逛著卡就刷下去了。買的時候都覺得好像可能會用到，但實際上好像根本就不會去讀，當擺設的實質意義好像遠大於書中的知識，比如那本被我拿來墊螢幕的JavaScript 大全以及被關著書櫃當造型的 CSS 大全 😅

說真的，我後來發現這種純教學得程式設計相關的書籍根本不太需要買，程式設計的知識更新太快了，一本出沒幾年的 vue 的書到現在其實有很多內容都算過時了，除非你完全不懂 vue 又剛好遇到那種稍微有點年紀的 vue 專案。對於這種純技術的工具書，網路資源其實更加方便，有很多影片和文件可以看，所以我以前買的這類書我要嘛送人要嘛就是關書櫃。 即使如此，我發現我還是超多本書沒讀。

最近篩選了一下沒讀但我覺得可能值得一讀的書我分成幾個種類：

  1. 技術經驗談：那些比較沒人系統整理的技術或網路資源較雜的知識。
  2. 自我成長類：我也不知為啥，累積一堆。
  3. 小說：身為三浦紫苑的粉絲，出新書不買是對她的不尊重 😃
